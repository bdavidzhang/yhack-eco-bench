Methodology

How we measure the environmental impact of LLM inference. Transparent, reproducible, ISO-standard.

The SCI Formula

SCI = (E × I) + M    per R
E — Energy

Energy per token in kWh. Measured via nvidia-smi power sampling at 1 Hz, integrated with the trapezoidal rule.

I — Carbon Intensity

Grid carbon intensity in gCO2/kWh. Default: Connecticut / ISO New England (~210 gCO2/kWh). Supports Electricity Maps API for real-time data.

M — Embodied Emissions

Manufacturing CO2 of the DGX Spark (~200 kg), amortized over a 5-year lifespan at average utilization, per functional unit.

R — Functional Unit

Per token generated. This makes SCI comparable across batch sizes and sequence lengths.

Hardware

NVIDIA DGX Spark
Blackwell GB10 GPU • ARM Cortex-X925 / A725 big.LITTLE CPU • 128 GB unified LPDDR5X (C2C) • 1 TB NVMe

Power Measurement

nvidia-smi sampled at 1 Hz (instantaneous + running average). Total energy computed via trapezoidal integration over the experiment duration. Sensor logger captures GPU temp, clocks, fan RPM, and NVMe temp at 1 s intervals.

Quality Metric

Validation bits-per-byte (BPB) on a held-out text dataset. Lower BPB = better language modeling quality. This is a standard compression-based metric used by scaling-law research.

Search Strategy

Grid search → Random → Bayesian optimization (Optuna). The workbench auto-switches strategies based on convergence. Pareto frontier updated after each experiment.

Thermal Safety

85 °C GPU abort threshold. Cooldown gating between experiments ensures thermal state doesn't bias power measurements. Thermal throttling is flagged in results.

Limitations

GPU power only (not full system draw). Embodied emissions estimated — NVIDIA has not published a GB10 lifecycle assessment. Grid intensity is static unless an Electricity Maps API key is configured.

ISO/IEC 21031:2024
Green Software Foundation — Software Carbon Intensity

The SCI specification is an ISO standard for measuring the carbon emissions of software systems. Our implementation follows the spec for per-request functional units in inference workloads.
